STEPS TO RUN THE PROJECT
STEP 1: 
Instal and run xampp Open XAMPP Control Panel
Start: ✅ Apache ✅ MySQL

If both turn green, you’re ready.

📁 STEP 2:
Place yourproject files Go to: C:\xampp\htdocs
Create a folder: eduhub Put ALL your files inside: eg index.php login.php, register.php, dashboard.php, students.php, teachers.php, courses.php admin.php logout.php config.php (or db.php)

STEP 3:
OPEN phpMyAdmin Open browser Go to: http://localhost/phpmyadmin/

STEP 4: 
CREATE DATABASE Click New Enter: eduhub Click Create

STEP 5: IMPORT / RUN SQL Click your database eduhub Click SQL tab Paste it in there

DATABASE IMPORT INSTRUCTION
Open PHPMYADMIN Set template name as database.sql

Then click save Then click import.
